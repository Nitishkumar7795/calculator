# Calculator Web Application

This is a simple web-based calculator that supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## How to Use
1. Open the `index.html` file in your web browser.
2. Use the on-screen buttons to enter numbers and operations.
3. Press `=` to see the result.
4. Press `C` to clear the display.

## Files
- `index.html`: Main HTML file
- `style.css`: Styles for the calculator
- `script.js`: Calculator logic
